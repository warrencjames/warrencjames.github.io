HTML Media Tags

The following tags are used to embed various media into a web page. Note that required attributes are listed in bold.

Tag Attributes Description
<img> src, alt, width, height Embeds an image into the document. Supports APNG, AVIF, GIF, JPEG, pNGSVG, and WebP. Self-closing tag. The "alt" attribute should be used to provide a text description for accessibility.
<video> src, height, width, autoplay, muted, loop, controls, preload Embeds a video into the document. "controls" attribute is required to enable simple controls. "src" attribute is optional for this tag, as the <source> tag with "src" and "type" attributes may be used within the block . Opening and closing tags are required. Text inside the video tags will appear if the video tag is not supported.
<audio> src, autoplay, muted, loop, controls, preload Embeds audio into the document. "controls attribute is required to enable simple controls. "src" attribute is optional for this tag, as the <source> tag with "src" and "type" attributes may be used within the block . Opening and closing tags are required. Text inside the audio tags will appear if the audio tag is not supported
<picture> (only global attributes) Embeds an image into the document with more flexibility in specifying image sources. The <source> tag may specify a list of image sources with the "srcset" attribute. The "media" attribute specifies a condition that will determine whether the browser will use it. An <img> tag should be used to provide a fallback in case the offered sources do not provide an image. It also describes the size and attributes of the image.
<source> src, type. (Picture only: srcset, media, height, width) Used within <audio>, <video>, and <picture> tags to specify the media's address and type.
